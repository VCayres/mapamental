OAB3D • Objeto Interativo: Conecte os Conceitos

Missão do estudante

Preencha as caixas com os conceitos aprendidos nas aulas anteriores, arraste para organizar e conecte as ideias em um mapa lógico próprio.

Como usar

  1. Crie caixas vazias.
  2. Digite os conceitos dentro delas.
  3. Ative Conectar e clique em duas caixas.
  4. Exporte em JSON para continuar depois.
  5. Use Imprimir PDF para gerar 1 página legível.
